# Heading

## Description

This component is similar to the Text component, but is larger than regular text.

## Props

```
children: string,
color: string ('black'),
decoration: oneOf(
  ['none']
),
bold: bool,
size: oneOf(
  ['xs', 'sm', 'md', 'lg', 'xl'] ('sm')
),
onPress: func,
align: string,
width: oneOfType(
  [string, number]
),
```
